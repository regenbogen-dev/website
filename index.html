アロナビ (AroNavi)

AroundMe Navigator

📱 アプリ概要

アロナビは、現在地から目的地への方向と距離をリアルタイムで表示する歩行者向けナビゲーションアプリです。地図上の方向矢印で直感的にナビゲーションを提供します。

主な機能

  • リアルタイム方向矢印ナビゲーション
  • あいまい検索・略称検索（コンビニ、カフェなど）対応
  • 地図の2D/3D表示切替
  • Apple Watch連携（独立GPS・コンパス対応）
  • 地図長押しによる直接目的地設定
  • 目的地到着時の自動案内終了
  • 検索履歴の管理

⚙️ 動作要件

iPhoneiOS 26.0 以降
Apple WatchwatchOS 26.0 以降（オプション）
位置情報「使用中のみ許可」が必要
通知Apple Watch連携使用時に推奨

❓ よくある質問

位置情報が取得できません

「設定」→「プライバシーとセキュリティ」→「位置情報サービス」からアロナビの位置情報アクセスを「使用中のみ」に設定してください。

Apple Watchに案内が表示されません

以下をご確認ください：

  • iPhoneとApple Watchがペアリングされていること
  • アロナビの設定で「Watchで案内を表示」がONになっていること
  • Watch側のアロナビアプリがインストールされていること

方向矢印が正しくありません

コンパスのキャリブレーション（8の字を描く動作）をお試しください。磁気の強い場所（電子機器の近くなど）では精度が低下する場合があります。

現在地が少し動きます

GPSの性質上、静止中でも数メートルの誤差が生じることがあります。Wi-FiをONにすると精度が向上します。

バッテリー消費について

案内中は高精度の位置情報を使用するため、通常より電力を消費します。長時間のナビゲーション時はバッテリー残量にご注意ください。

🔒 プライバシーポリシー

アロナビは以下のデータを使用します：

  • 位置情報：ナビゲーションの提供およびマップ表示のためにのみ使用します。
  • 検索履歴：お使いのデバイス内にのみ保存されます。

当アプリは、個人データを外部サーバーに送信したり、第三者と共有したりすることはありません。すべてのデータはお使いのデバイス上で処理されます。

地図データおよび検索結果はApple MapKitを通じて提供されます。これらのデータの取り扱いについてはAppleのプライバシーポリシーに準じます。

📱 About the App

AroNavi is a pedestrian navigation app that displays real-time direction and distance from your current location to your destination. It provides intuitive navigation with directional arrows on the map.

Key Features

  • Real-time directional arrow navigation
  • Fuzzy search and abbreviation support
  • 2D/3D map view toggle
  • Apple Watch support (independent GPS & compass)
  • Long-press to set destination directly on map
  • Automatic arrival detection
  • Search history management

❓ FAQ

Location is not being detected

Go to Settings → Privacy & Security → Location Services, and set AroNavi's location access to "While Using the App."

Navigation doesn't appear on Apple Watch

Please check the following:

  • iPhone and Apple Watch are paired
  • "Show navigation on Watch" is enabled in AroNavi settings
  • AroNavi Watch app is installed

Direction arrow seems inaccurate

Try calibrating the compass by moving your device in a figure-8 pattern. Accuracy may decrease near strong magnetic fields (e.g., near electronic devices).

My location drifts slightly when stationary

GPS inherently has a few meters of error even when stationary. Turning on Wi-Fi can improve accuracy.

Battery consumption

High-accuracy location is used during navigation, which consumes more power than usual. Please monitor your battery level during extended navigation.

🔒 Privacy Policy

AroNavi uses the following data:

  • Location data: Used solely for navigation and map display.
  • Search history: Stored only on your device.

This app does not transmit personal data to external servers or share it with third parties. All data is processed on your device.

Map data and search results are provided through Apple MapKit. Please refer to Apple's privacy policy for the handling of this data.
